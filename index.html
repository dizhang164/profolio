3D作品：電臀美樂蒂

Monster Mash

【作品簡介】
這是一隻正在盡情「電臀」的美樂蒂，充滿活力與趣味！

【創作心得】
這個作品是使用 Monster Mash 軟體完成的。當初在選擇主題時，因為覺得美樂蒂這個玩偶超級可愛，加上想要做點有動態反差感的效果，於是就誕生了這個充滿魔性的 3D 動畫。看著平面的塗鴉瞬間變成能動的立體玩偶，過程真的超療癒。

學習反思與總結

這門科技應用專題課讓我收穫滿滿，課程主要包含四大核心探索：

  1. 使用 Monster Mash 及 Polycam 製作 3D 物件。
  2. 使用 Google AI Studio 生成互動式網頁遊戲。
  3. 使用 GitHub 進行網頁的雲端部署與上傳。
  4. 使用 Gemini 協作生成這個充滿科技感的學習成果首頁。

【心得總結】
在所有環節中，製作 3D 物件的過程讓我感到非常有趣；而 Google AI Studio 能夠直接生成網頁遊戲的強大能力，更是令我印象深刻。

不過，這趟旅程也有充滿挑戰的時刻。我覺得最困難的就是將網頁上傳到 GitHub 的過程。步驟非常瑣碎，途中碰到了非常多的難關與錯誤訊息。還好我沒有放棄，將問題一一解決，最後順利看見自己的心血上線，這份滿滿的成就感是我這堂課最大的收穫！
